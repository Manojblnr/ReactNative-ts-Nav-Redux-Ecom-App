
1. Setup react native project

2. Run the application:
    1. Build in android studio
    2. adb reverse tcp:8081 8081
    3. npm start    
    4. run the app  


3. Install Naviagtion dependencies
    Link: https://reactnavigation.org/docs/getting-started

    navigation:
        1. npm install @react-navigation/native
        2. npm install @react-navigation/bottom-tabs
        3. npm install @react-navigation/native-stack
    
    Icons:
        1. npm install @types/react-native-vector-icons
        2. npm install react-native-vector-icons
    
4.  Create src folder and index.tsx, Somponents, Screens, Types, Naviagtion
    RootNavigator.tsx, TabNavigator.tsx folder
